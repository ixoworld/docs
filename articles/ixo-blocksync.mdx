<Note>
  Blocksync provides real-time indexing and querying of IXO blockchain data through GraphQL, enabling efficient access to historical transactions, state changes, and entity relationships.
</Note>

## Overview

Blocksync serves as a critical infrastructure component that:
- Indexes blockchain data in real-time
- Provides fast GraphQL querying
- Maintains historical state
- Enables complex data relationships

## Query Examples

<CodeGroup>
  ```graphql Entity Query
  query GetEntity {
    entity(id: "did:ixo:123") {
      id
      type
      data
      claims {
        id
        type
        status
      }
      verifications {
        id
        status
        verifier
      }
    }
  }
  ```

  ```graphql Claims Query
  query GetClaims {
    claims(
      filter: {
        type: "impact",
        status: "approved",
        fromDate: "2023-01-01"
      }
    ) {
      id
      type
      data
      evaluations {
        id
        status
        evaluator
      }
    }
  }
  ```

  ```graphql Verification Query
  query GetVerifications {
    verifications(
      filter: {
        type: "oracle",
        status: "completed"
      }
    ) {
      id
      type
      result
      proof
      timestamp
    }
  }
  ```
</CodeGroup>

## Integration Examples

<CodeGroup>
  ```typescript TypeScript
  import { BlocksyncClient } from '@ixo/blocksync-client';

  const client = new BlocksyncClient({
    endpoint: 'https://blocksync.emerging.eco/graphql',
    headers: {
      'Authorization': `Bearer ${apiKey}`
    }
  });

  const result = await client.query({
    query: GET_ENTITY,
    variables: { id: 'did:ixo:123' }
  });
  ```

  ```python Python
  from ixo_blocksync import BlocksyncClient

  client = BlocksyncClient(
      endpoint='https://blocksync.emerging.eco/graphql',
      api_key='your-api-key'
  )

  result = client.query(
      query=GET_ENTITY,
      variables={'id': 'did:ixo:123'}
  )
  ```

  ```javascript JavaScript
  import { createClient } from '@ixo/blocksync-client';

  const client = createClient({
    url: 'https://blocksync.emerging.eco/graphql',
    headers: {
      'Authorization': `Bearer ${apiKey}`
    }
  });

  const { data } = await client.query({
    query: GET_ENTITY,
    variables: { id: 'did:ixo:123' }
  });
  ```
</CodeGroup>
