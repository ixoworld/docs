## Blockchain Interfaces

<Note>
  The IXO blockchain provides three primary interfaces for interacting with nodes: CLI, gRPC, and REST. While all interfaces support state queries, only CLI can generate and sign transactions directly.
</Note>

<AccordionGroup>
  <Accordion title="Command-Line Interface (CLI)" icon="terminal">
    The CLI provides direct node interaction through the `ixod` binary, which serves as both node and application client.

    ```bash
    # View all commands
    ixod --help

    # View transaction commands
    ixod tx --help

    # View query commands
    ixod query --help
    ```

    The CLI is particularly useful for:
    - Node management
    - Transaction generation and signing
    - State queries
    - Network configuration
  </Accordion>

  <Accordion title="gRPC Interface" icon="server">
    gRPC enables programmatic blockchain interaction using protocol buffers.

    ```bash
    # List available services
    grpcurl -plaintext localhost:9090 list

    # Query account balances
    grpcurl \
        -plaintext \
        -d '{"address":"<address>"}' \
        localhost:9090 \
        cosmos.bank.v1beta1.Query/AllBalances
    ```

    Key features:
    - High-performance RPC framework
    - Strong typing with protocol buffers
    - Language-agnostic interface
    - Streaming capabilities
  </Accordion>

  <Accordion title="REST Interface" icon="globe">
    REST endpoints are available through gRPC Gateway, providing HTTP access to all gRPC services.

    ```bash
    # Query account balances
    curl \
        -X GET \
        -H "Content-Type: application/json" \
        http://localhost:1317/cosmos/bank/v1beta1/balances/<address>
    ```

    Benefits include:
    - Familiar HTTP/REST paradigm
    - OpenAPI/Swagger documentation
    - Easy integration with web applications
    - Browser-friendly interface
  </Accordion>
</AccordionGroup>

## Integration Guide

<CodeGroup>
  ```typescript Node Integration
  import { IxoClient } from '@ixo/client-sdk';

  const client = new IxoClient({
    rpcUrl: 'https://rpc.emerging.eco',
    chainId: 'impact-hub-1'
  });

  // Create and sign transaction
  const tx = await client.createTx({
    messages: [createClaimMsg],
    memo: 'Create impact claim'
  });

  const result = await client.signAndBroadcast(tx);
  ```

  ```python Python Integration
  from ixo_sdk import IxoClient

  client = IxoClient(
      rpc_url='https://rpc.emerging.eco',
      chain_id='impact-hub-1'
  )

  // Create and sign transaction
  tx = client.create_tx(
      messages=[create_claim_msg],
      memo='Create impact claim'
  )

  result = client.sign_and_broadcast(tx)
  ```

  ```go Go Integration
  import "github.com/ixofoundation/ixo-sdk-go"

  client := ixo.NewClient(
      "https://rpc.emerging.eco",
      "impact-hub-1",
  )

  // Create and sign transaction
  tx := client.CreateTx(
      []sdk.Msg{createClaimMsg},
      "Create impact claim",
  )

  result := client.SignAndBroadcast(tx)
  ```
</CodeGroup>
