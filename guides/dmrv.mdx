## Integration Example

<CodeGroup>

```python
from emerging import Device, Oracle, Credential

# Initialize device and oracle
device = Device("did:ixo:device/123")
oracle = Oracle("did:ixo:oracle/456")

# Collect and verify measurements
measurements = device.collect_measurements()
verification = oracle.verify_measurements(measurements)

# Issue verified credential
if verification.is_valid:
    credential = Credential.issue(
        type="MeasurementClaim",
        subject=measurements,
        evidence=verification.proof
    )
```

```javascript
import { Device, Oracle, Credential } from '@emerging/sdk';

// Initialize device and oracle
const device = new Device('did:ixo:device/123');
const oracle = new Oracle('did:ixo:oracle/456');

// Collect and verify measurements
const measurements = await device.collectMeasurements();
const verification = await oracle.verifyMeasurements(measurements);

// Issue verified credential
if (verification.isValid) {
  const credential = await Credential.issue({
    type: 'MeasurementClaim',
    subject: measurements,
    evidence: verification.proof
  });
}
```
