## Prerequisites

- Install [IXO SDKs](/sdk-reference/index)
- Set up [Signing Client](https://www.npmjs.com/package/@ixo/impactxclient-sdk#signing-client) for transaction signing
- Access to [IXO Blocksync GraphQL](/api-reference/blocksync-graphql-api) for queries

<Tip>
Signing Client is required for transaction broadcasting when using the IXO MultiClient SDK.
</Tip>

## Quick Start

### Entity Queries
Query entities through the Blocksync API:

```graphql
query GetEntity($id: String!) {
  entity(id: $id) {
    id
    type
    status
  }
}
```

### Available Data Queries
- Claims and Collections
- Tokens and Swaps  
- Bonds and Entities
- Chain Information
- Transaction History
