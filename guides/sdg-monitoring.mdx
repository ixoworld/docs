## Quick Start

<CodeGroup>

```bash Submit SDG Claim
curl -X POST https://api.emerging.eco/v1/claims \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "type": "SDGClaim",
    "indicator": "SDG1",
    "householdId": "did:ixo:household/123",
    "measurement": {
      "fuelSavings": 50.00,
      "currency": "USD"
    }
  }'
```

```python
from emerging import Client

client = Client('YOUR_API_KEY')
claim = client.claims.create(
    type="SDGClaim",
    indicator="SDG1",
    household_id="did:ixo:household/123",
    measurement={
        "fuel_savings": 50.00,
        "currency": "USD"
    }
)
```

</CodeGroup>

## Required Parameters

<ResponseField name="type" type="string" required>
  Type of SDG claim (e.g., "SDGClaim")
</ResponseField>

<ResponseField name="indicator" type="string" required>
  SDG indicator code
</ResponseField>

<ResponseField name="householdId" type="string" required>
  DID of the household
</ResponseField>

<ResponseField name="measurement" type="object" required>
  Indicator-specific measurements
</ResponseField>

## Validation Rules

### SDG 1: Poverty Reduction

```json
{
  "measurement": {
    "fuelSavings": {
      "type": "number",
      "minimum": 0,
      "description": "Annual fuel cost savings in USD"
    }
  }
}
```

### SDG 7: Clean Energy Access

```json
{
  "measurement": {
    "cleanCookingUsage": {
      "type": "number",
      "minimum": 0,
      "maximum": 100,
      "description": "Percentage of cooking done with clean fuels"
    }
  }
}
```

## Oracle Integration

<CodeGroup>

```python Validate Claim
# Initialize Oracle
oracle = SDGOracle(credentials="YOUR_ORACLE_KEY")

# Validate measurement
result = oracle.validate_claim({
    "type": "SDG7",
    "measurement": {
        "clean_cooking_usage": 85
    }
})
```

```javascript
// Initialize Oracle
const oracle = new SDGOracle({
  credentials: 'YOUR_ORACLE_KEY'
});

// Validate measurement
const result = await oracle.validateClaim({
  type: 'SDG7',
  measurement: {
    cleanCookingUsage: 85
  }
});
```

</CodeGroup>
