## Common Patterns

The SDK follows consistent patterns for blockchain interactions:

1. Create a client instance
2. Get user accounts/addresses
3. Construct message with appropriate type URL
4. Sign and broadcast transaction
5. Return response

<Tip type="info">
For detailed API documentation and examples, refer to the [IXO MultiClient SDK module reference](https://github.com/ixofoundation/ixo-multiclient-sdk).
</Tip>
