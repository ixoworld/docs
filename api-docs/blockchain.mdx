## Core Endpoints

### Transaction Endpoints

```typescript
POST /txs
GET /txs/{hash}
GET /txs/search
```

### Query Endpoints

```typescript
GET /blocks/latest
GET /bank/balances/{address}
GET /auth/accounts/{address}
GET /staking/validators
```

### Smart Contract Endpoints

```typescript
GET /wasm/code/{code_id}
GET /wasm/contract/{contract_address}
POST /wasm/contract/{contract_address}/execute
```

## Implementation Guide

<Steps>
  1. **Setup Client**
     ```typescript
     import { IxoClient } from '@ixo/client'
     
     const client = new IxoClient({
       rpcUrl: 'https://rpc.ixo.earth',
       chainId: 'ixo-4'
     })
     ```

  2. **Create Transaction**
     ```typescript
     const tx = await client.createTx({
       messages: [/* transaction messages */],
       memo: 'Optional memo',
       fee: {
         amount: [{ amount: '5000', denom: 'uixo' }],
         gas: '200000'
       }
     })
     ```

  3. **Sign & Broadcast**
     ```typescript
     const signedTx = await client.sign(tx, signer)
     const result = await client.broadcast(signedTx)
     ```

  4. **Query State**
     ```typescript
     const balance = await client.getBalance(address)
     const validator = await client.getValidator(valAddress)
     ```
</Steps>
