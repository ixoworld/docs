<Note>
  The ITMO Schema defines the structure and validation rules for Internationally Transferred Mitigation Outcomes (ITMOs) as Verifiable Credentials. This standardized format ensures interoperability and compliance with Article 6.2 of the Paris Agreement.
</Note>

## Schema Structure

<AccordionGroup>
  <Accordion title="Context and Type">
    ```json
    {
      "@context": [
        "https://www.w3.org/2018/credentials/v1",
        "https://w3id.org/security/suites/ed25519-2018/v1",
        {
          "itmo": "https://w3id.org/article6/itmo-context.jsonld",
          "prov": "http://www.w3.org/ns/prov#"
        }
      ],
      "type": [
        "VerifiableCredential",
        "ITMOCredential"
      ]
    }
    ```
  </Accordion>

  <Accordion title="Credential Metadata">
    ```json
    {
      "id": "urn:uuid:<uuid>",
      "issuer": "did:ixo:entity:<issuer-id>",
      "issuanceDate": "<ISO-8601-date>",
      "expirationDate": "<ISO-8601-date>"
    }
    ```
  </Accordion>
</AccordionGroup>

## Credential Attributes

<AccordionGroup>
  <Accordion title="Authorization Info" icon="key">
    ```json
    {
      "itmo:authorizationInfo": {
        "itmo:authorizationReference": "string",
        "itmo:cooperativeApproachDuration": "ISO-8601-interval",
        "itmo:expectedMitigationOutcomes": {
          "year": "number"
        },
        "itmo:participatingParties": ["string"],
        "itmo:authorizedEntities": ["did:ixo:entity:id"]
      }
    }
    ```
    
    The authorization information includes:
    - **authorizationReference**: Unique reference for the authorization
    - **cooperativeApproachDuration**: Time period for the cooperative approach
    - **expectedMitigationOutcomes**: Annual mitigation targets
    - **participatingParties**: List of participating countries
    - **authorizedEntities**: List of authorized entity DIDs
  </Accordion>

  <Accordion title="NDC Quantification" icon="calculator">
    ```json
    {
      "itmo:ndcQuantification": {
        "itmo:ndcSectors": ["string"],
        "itmo:ndcSources": ["string"],
        "itmo:ndcGHGs": ["string"],
        "itmo:ndcTimePeriods": ["string"],
        "itmo:ndcReferenceLevel": "number",
        "itmo:ndcTargetLevel": "number",
        "itmo:ndcQuantificationTonnesCO2e": "number"
      }
    }
    ```
    
    Quantification metrics include:
    - **ndcSectors**: Affected economic sectors
    - **ndcSources**: Emission source categories
    - **ndcGHGs**: Greenhouse gases covered
    - **ndcTimePeriods**: Relevant time periods
    - **ndcReferenceLevel**: Baseline emissions level
    - **ndcTargetLevel**: Target emissions level
    - **ndcQuantificationTonnesCO2e**: Quantified impact
  </Accordion>

  <Accordion title="Corresponding Adjustments" icon="scale-balanced">
    ```json
    {
      "itmo:correspondingAdjustments": {
        "itmo:singleYearNDC": "boolean",
        "itmo:multiYearNDC": "boolean",
        "itmo:emissionsTrajectory": ["number"],
        "itmo:annualCorrespondingAdjustments": ["number"],
        "itmo:averageAnnualITMOs": "number",
        "itmo:cumulativeAdjustments": "number"
      }
    }
    ```
    
    Adjustment details include:
    - **singleYearNDC**: Single year target indicator
    - **multiYearNDC**: Multi-year target indicator
    - **emissionsTrajectory**: Projected emissions path
    - **annualCorrespondingAdjustments**: Year-by-year adjustments
    - **averageAnnualITMOs**: Average annual transfers
    - **cumulativeAdjustments**: Total cumulative adjustments
  </Accordion>

  <Accordion title="Environmental Integrity" icon="leaf">
    ```json
    {
      "itmo:environmentalIntegrity": {
        "itmo:environmentalIntegrityMetrics": ["string"],
        "itmo:ipccMethodologiesReference": "string",
        "itmo:partyStatusUnderParis": "boolean",
        "itmo:maintainedNDC": "boolean",
        "itmo:trackingArrangements": ["string"],
        "itmo:mostRecentNationalInventory": "string",
        "itmo:contributionToNDCImplementation": "boolean"
      }
    }
    ```
    
    Integrity validation includes:
    - **environmentalIntegrityMetrics**: Measurement metrics used
    - **ipccMethodologiesReference**: IPCC methodology reference
    - **partyStatusUnderParis**: Paris Agreement compliance
    - **maintainedNDC**: NDC maintenance status
    - **trackingArrangements**: Monitoring arrangements
    - **mostRecentNationalInventory**: Latest inventory reference
    - **contributionToNDCImplementation**: NDC contribution status
  </Accordion>
</AccordionGroup>
