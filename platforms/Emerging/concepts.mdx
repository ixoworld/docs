<Tip>
The Emerging Household Energy Platform enables intelligent coordination, financing, verification, governance, and data collection for climate impact mitigation, using blockchain, IoT, and AI technologies, with a focus on clean cooking and household energy solutions.
</Tip>

## Digital MRV System

The platform provides a comprehensive decentralized Measurement, Reporting, and Verification (dMRV) system that offers:

<AccordionGroup>

<Accordion title="Real-Time Monitoring">
- **IoT Integration**: Smart device connectivity and data streaming
- **Continuous Monitoring**: Real-time usage and performance tracking
- **Standardized Reporting**: Structured data collection and validation
- **Secure Validation**: Cryptographic proofs and verification
</Accordion>

<Accordion title="Automated Verification">
- **Protocols**: Methodology alignment and reporting
- **Claims Processing**: Data validation and evaluation
- **Automated Checks**: Data completeness and threshold compliance
- **Oracle Services**: Causal AI-powered validation and evaluation
- **Manual Reviews**: Expert validation and audit
</Accordion>

<Accordion title="Digital Certification">
- **Verifiable Credentials**: Standardized digital certificates
- **ITMO Certificates**: Digital carbon credit certification
- **Selective Disclosure**: Privacy-preserving verification
</Accordion>

</AccordionGroup>

## Implementation Guide

<AccordionGroup title="Implementation Steps" icon="list-check">
<Accordion title="Domain Setup">
<Steps>
- Register Entity Domains for Projects, Households, and Devices
- Register User DIDs with Mobile Wallets
- Issue Veriable Credentials for households and devices
- Configure Protocols, or select existing ones
</Steps>
</Accordion>

<Accordion title="Data Collection">
<Steps>
- Configure Claim Collections, using Protocols
- Authorize Agents to submit Claims
- Connect IoT devices to Matrix API
- Submit Veriable Claims
</Steps>
</Accordion>

<Accordion title="Verification Process">
<Steps>
- Set up automated checks
- Configure oracle services
- Validate with manual reviews
- Generate verification reports
- Audit with cryptographic proofs
</Steps>
</Accordion>

<Accordion title="Market Integration">
<Steps>
- Connect to marketplaces
- Enable credit trading
- Set up payment systems
- Integrate with registries

</Steps>
</Accordion>
</AccordionGroup>
