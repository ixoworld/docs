## Quick Start

<CodeGroup>

```bash Install SDKs
# Install core SDKs
npm install @ixo/impactxclient-sdk @ixo/matrixclient-sdk
npm install @ixo/oracle-agent-sdk @ixo/signx-sdk
```

```typescript Initialize Client
import { createClient } from '@ixo/impactxclient-sdk';

const client = await createClient({
  chainId: 'ixo-5',
  endpoint: 'https://rpc.ixo.earth'
});
```

</CodeGroup>

## Core Features

### Digital Domains

```typescript
// Create digital domain
const domain = await client.entity.create({
  type: "DigitalDomain",
  controller: ownerAddress,
  metadata: {
    name: "Example Domain"
  }
});
```

### Secure Data Rooms

```typescript
// Create encrypted room
const room = await matrix.createRoom({
  name: "Domain Data",
  encryption: true,
  members: ["@alice:ixo.world"]
});
```

### Oracle Verification

```typescript
// Verify data
const verification = await oracle.verify({
  type: "DataVerification",
  subject: domain.did,
  data: domainData
});
```
