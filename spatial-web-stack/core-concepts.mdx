# Spatial Web Stack

<Tip>
  The IXO Spatial Web Stack is a comprehensive suite of technologies that enable the creation of cognitive digital twin systems, secure data management, and verifiable claims through blockchain integration, decentralized data storage, and Agentic Oracles.
</Tip>

## Getting Started

<Steps>
  <Step title="Understand the Components">
    Familiarize yourself with the core components of the Spatial Web Stack and how they work together.
  </Step>
  
  <Step title="Set Up Development Environment">
    Install the necessary SDKs and tools for your development environment.
    
    ```bash
    # Install core SDKs
    npm install @ixo/impactxclient-sdk @ixo/matrixclient-sdk
    npm install @ixo/oracle-agent-sdk
    ```
  </Step>
  
  <Step title="Build Your Application">
    Leverage the SDKs to implement your application's core functionality:
    - Create and manage digital twins
    - Set up secure data rooms
    - Implement verification workflows
    - Integrate with wallets and credentials
  </Step>
  
  <Step title="Deploy and Monitor">
    Deploy your application to production and set up monitoring and maintenance processes.
  </Step>
</Steps>
