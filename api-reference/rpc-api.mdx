## Entity Operations

### Create Entity

<CodeGroup>

```json Request
POST https://rpc.ixo.earth/txs
{
  "type": "cosmos-sdk/MsgCreateEntity",
  "value": {
    "creator": "ixo1...",
    "entityType": "asset",
    "entityStatus": 1
  }
}
```

```json Response
{
  "height": "123456",
  "txhash": "0xABC...",
  "data": {
    "entityId": "did:ixo:entity/123"
  }
}
```

</CodeGroup>

### Query Entity

```bash
curl https://rpc.ixo.earth/ixo/entity/v1beta1/entity/{did}
```

<ResponseField name="did" type="string" required>
  Entity's decentralized identifier
</ResponseField>

## Claims Processing

<CodeGroup>

```json Submit Claim
POST https://rpc.ixo.earth/txs
{
  "type": "cosmos-sdk/MsgSubmitClaim",
  "value": {
    "creator": "ixo1...",
    "claimId": "claim123",
    "claimType": "verification"
  }
}
```

```json Evaluate Claim
POST https://rpc.ixo.earth/txs
{
  "type": "cosmos-sdk/MsgEvaluateClaim",
  "value": {
    "claimId": "claim123",
    "status": "approved"
  }
}
```

</CodeGroup>
