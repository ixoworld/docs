<Tip>
The IXO RPC API provides protobuf-based interfaces for interacting with the blockchain, defining key message types and services for transactions, events, and state queries.
</Tip>

## Example Messages

### Create Entity
```protobuf
message MsgCreateEntity {
  string creator = 1;
  string entity_type = 2;
  string entity_status = 3;
}
```

### Submit Claim
```protobuf
message MsgSubmitClaim {
  string creator = 1;
  string collection_id = 2;
  string claim_id = 3;
}
```
