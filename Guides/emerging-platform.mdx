## Overview

<Note>
  The Emerging Platform enables developers to create, track, and verify climate impact initiatives using blockchain and IoT technologies, with machine-readable verifiable data and AI optimisation.
</Note>

## Quick Start

<CodeGroup>
  ```bash Install SDK
  # Install via npm
  npm install @emerging/sdk

  # Install via pip
  pip install emerging-sdk
  ```

  ```javascript Initialize Client
  import { EmergingClient } from '@emerging/sdk';

  const client = new EmergingClient({
    apiKey: 'YOUR_API_KEY',
    environment: 'testnet' // or 'mainnet'
  });
  ```

  ```python
  from emerging import Client

  client = Client(
      api_key='YOUR_API_KEY',
      environment='testnet'  # or 'mainnet'
  )
  ```
</CodeGroup>

## Core Concepts

<AccordionGroup>
  <Accordion title="Digital Identifiers (DIDs)">
    * Unique identifiers for households, devices, and organizations

    * Cryptographically verifiable ownership

    * Basis for all platform interactions
  </Accordion>

  <Accordion title="Claims & Credentials">
    * Submit data as verifiable claims

    * Receive validated credentials for digital certificates

    * Build trust through cryptographic proofs, transparency and auditability
  </Accordion>

  <Accordion title="Oracle Services">
    * Automated claims processing and digital certification

    * External data integration and validation

    * Intelligent workflow automations, with humans in the loop

    * Advanced data queries and analytics

    * Intelligent decision-support using causal logic
  </Accordion>
</AccordionGroup>

## Authentication

All API requests require an API key:

```bash
Authorization: Bearer YOUR_API_KEY
```

<Note>
  Get your API key from the [Emerging Portal](https://portal.emerging.eco).
</Note>
