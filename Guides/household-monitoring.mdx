## Digital Twin Architecture

<CodeGroup>
  ```json Household Twin
  {
    "did": "did:ixo:household/123",
    "type": "CleanCookingHousehold",
    "location": {
      "district": "XYZ",
      "density": "Medium"
    },
    "devices": [{
      "did": "did:ixo:device/456",
      "type": "ElectricPressureCooker",
      "status": "active"
    }],
    "baseline": {
      "fuelType": "Charcoal",
      "monthlyConsumption": 40
    }
  }
  ```

  ```python Create Household
  from emerging import Household, Device

  # Create household digital twin
  household = Household.create(
      location={
          "district": "XYZ",
          "density": "Medium"
      },
      baseline={
          "fuel_type": "Charcoal",
          "monthly_consumption": 40
      }
  )

  # Add device to household
  device = Device.get("did:ixo:device/456")
  household.add_device(device)
  ```
</CodeGroup>

### Stove Stacking Analysis

```python
from emerging import StoveStackingOracle

# Initialize oracle
oracle = StoveStackingOracle()

# Analyze household cooking patterns
analysis = oracle.analyze_household(
    household_id="did:ixo:household/123",
    period_start="2024-01-01",
    period_end="2024-01-31"
)

# Get stove stacking metrics
metrics = analysis.get_metrics()
print(f"Primary stove usage: {metrics.primary_stove_percentage}%")
print(f"Secondary stove usage: {metrics.secondary_stove_percentage}%")
```

## Credential Issuance

<CodeGroup>
  ```json Household Credential
  {
    "@context": [
      "https://www.w3.org/2018/credentials/v1",
      "https://w3id.org/household-monitoring/v1"
    ],
    "type": ["VerifiableCredential", "HouseholdImpactCredential"],
    "issuer": "did:ixo:oracle/789",
    "credentialSubject": {
      "id": "did:ixo:household/123",
      "period": {
        "start": "2024-01-01",
        "end": "2024-01-31"
      },
      "impacts": {
        "emissionReductions": 75,
        "fuelSavings": 32,
        "costSavings": 45
      },
      "evidence": [{
        "type": "DeviceTelemetry",
        "deviceId": "did:ixo:device/456",
        "dataHash": "0x123..."
      }]
    }
  }
  ```

  ```python Issue Credential
  from emerging import Credential, Evidence

  # Compile evidence
  evidence = Evidence.from_telemetry(
      device_id="did:ixo:device/456",
      period_start="2024-01-01",
      period_end="2024-01-31"
  )

  # Issue household impact credential
  credential = Credential.issue(
      type="HouseholdImpactCredential",
      subject_id="did:ixo:household/123",
      impacts={
          "emission_reductions": 75,
          "fuel_savings": 32,
          "cost_savings": 45
      },
      evidence=[evidence]
  )
  ```
</CodeGroup>

## API Integration

### Household Management

```bash
# Create household
curl -X POST https://api.emerging.eco/v1/households \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "location": {
      "district": "XYZ",
      "density": "Medium"
    },
    "baseline": {
      "fuelType": "Charcoal",
      "monthlyConsumption": 40
    }
  }'
```

### Impact Analysis

```bash
# Analyze household impact
curl -X POST https://api.emerging.eco/v1/analysis/household \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "householdId": "did:ixo:household/123",
    "periodStart": "2024-01-01",
    "periodEnd": "2024-01-31"
  }'
```
